# Victrl

## — Plug-and-Play HID AI Agent

[[中文](README_CN.md)|English]

Current AI Agents rely on basic software configuration on target devices to control them. Victrl aims to build a pure hardware AI Agent device independent of the operating systems of controlled devices. It achieves plug-and-play automated operation on any device in a human-like manner via visual input and HID output.

Victrl does not rely on OS APIs, Accessibility, ADB, VNC, RDP and other protocols. Instead, it fully simulates the process of humans operating computers:

```
analyzing screen content → controlling keyboard and mouse → observing execution results.
```

Essentially, this project explores a brand-new architecture for Hardware AI Agent.
